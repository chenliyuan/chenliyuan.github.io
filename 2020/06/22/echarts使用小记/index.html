  12. 图像链接跳转重复打开多个浏览器tab

    1 使用windows.open 容易出现重复打开窗口的情况；若确保每次都为其提供相同的窗口名称可解决，由于项目中窗口名称不太好取，就采用了下面的方式

    export function openNewurl(url) {
  var newA = document.createElement("a");
  newA.id = 'gg'
  newA.target = '_blank';
  newA.href = url;
  document.body.appendChild(newA);
  newA.click();
  document.body.removeChild(newA);
}

    2 但是还有重复的情况，排查知进行了多次click事件，原因是进行了重复绑定，需要进行解绑，进行off

    参考：https://github.com/apache/incubator-echarts/issues/6089
