JS 事件基础和进阶

Posted on 2021-02-20
事件调用方式

  • 在script标签中调用

    obj.事件=function(){}

  • 在元素中调用

常见事件有五种：
  • 鼠标事件
  • 键盘事件
  • 表单事件
  • 编辑事件
  • 页面事件
鼠标事件
onclick
onmouseover
onmouseout
onmousedown
onmouseup
onmousemove
键盘事件
onkeydown
onkeyup
表单事件
onfocus/onblur //获取焦点和失去焦点 onfocus属性区别于focus()方法
onselect//区别于select()方法
onchange
编辑事件

主要用于限制复制操作

oncopy
onselectstart//选中
oncontextmenu //右键菜单
页面事件
onload//文档下载完成后再执行的一个事件
window.onload=function(){}
onbeforeunload//离开页面之前的一个事件
事件处理器和监听器
oBtn.onclick=function(){}//事件处理器
obj.addEventListener(type,fn,false)
obj.removeEventListner(type,fn,false)
event事件
每个事件都有对应的event对象
event对象属性：
type
keyCode
shiftKey
ctrlKey
altKey
this

哪个DOM对象调用了this所在的函数，那么this指向的就是哪个DOM对象

this只有在被调用的时候才确定下来
